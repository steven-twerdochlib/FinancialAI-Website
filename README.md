# FinancialAI Vue

A Vue 3 project designed to promote FinancialAI company, team and products.

## Quick start

1. Install dependencies
```bash
npm install
```

2. Run dev server
```bash
npm run dev
```

3. Build and release for production
```bash
npm run deploy
```

The hero image used is included at `src/assets/hero-spreadsheet.png`.
